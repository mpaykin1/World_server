## World-Sharabass (миршарабас)
Файл: `apps/world-sharabass/index.html`

| Элемент | Описание |
|---------|----------|
| **Пол** | `GridHelper(60, 30)` — сетка 60×60 с ячейками 2×2, на y=-1.49 (уровень SDF пола) |
| **Персонаж** | Модель из каталога (капсула + голова с лицом + руки/ноги коробками) на y=-1.5 |
| **Диск** | Голубая полупрозрачная окружность под ногами персонажа |
| **Подсветка THREE.js** | AmbientLight(0.7) + DirectionalLight(1.4) |
| **Стройка** | B / кнопка "🚧 Строить": призрак предмета на ближайшей ячейке сетки (2×2) в 2м перед персонажем |
| **Установка** | Space / Enter / клик / тап — объект ставится на ячейку сетки |

### Управление
| Действие | PC | Touch |
|----------|----|-------|
| Ходьба | WASD / стрелки | 1 палец drag |

| Вид от 1-го лица | Q | — |
| Стройка | B | кнопка 🚧 |
| Поставить объект | Space / клик | тап |
| Размер (стройка) | Q/E | — |
| Музыка | любая клавиша / клик | тап (второй и далее) |

| Компактность | Весь UI уменьшен на ~40% (`scale(.6)`) и прозрачность `.7` | — |
