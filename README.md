# WebGL Survival Hub v3 NO-NPM

Один сервер для всех браузерных приложений/игр: 3D-каталог, общая система аккаунтов, общий глобальный чат и survival-игра.

Эта версия **не требует `npm install`**. Сервер работает на встроенных модулях Node.js и своём мини-WebSocket сервере.

## Запуск

```powershell
node server.js
```

или:

```powershell
npm start
```

Открыть:

```text
http://localhost:3000
```

## Архитектура приложений

Каждое приложение лежит в отдельной папке:

```text
apps/
  catalog/
    ico.png
    client.js
    index.html
  survival/
    ico.png
    client.js
    index.html
  chat/
    ico.png
    index.html
```

## Важно: порталы теперь автоматические

`server.js` сам анализирует папку `apps/`.

Если ты добавишь новую папку:

```text
apps/my_game/
  ico.png
  client.js
  index.html
```

то `/api/apps` сразу увидит приложение, а 3D-каталог автоматически создаст для него портал. Ничего вручную в коде каталога переписывать не нужно даже для 200 приложений.

Название портала берётся из `<title>` внутри `index.html`. Если title нет, берётся имя папки.

## Что исправлено в v3

### 1. Инвентарь как в Minecraft

Теперь есть два уровня:

- постоянный hotbar снизу на 9 слотов;
- большой экран Inventory по центру;
- 36 слотов всего;
- нижние 9 слотов являются hotbar;
- `1-9` выбирают слот hotbar;
- `I` или `E` открывают/закрывают Inventory.

### 2. Строительство исправлено

Стройка стала ближе к Rust-style Building Plan:

- `B` включает режим строительства;
- зелёный ghost preview показывает место установки;
- красный ghost preview показывает неправильное место;
- фундамент ставится по сетке;
- стены и doorway теперь снапаются к краям фундамента;
- дверь ставится только в doorway;
- лестница снапается к ближайшему фундаменту;
- сервер проверяет дистанцию, ресурсы, cooldown, коллизии и наличие опоры.

### 3. Лицо персонажа

У игрока теперь есть отдельное лицо: глаза, нос, рот и брови. Добавлено и в survival, и в 3D-каталог.

## Управление в каталоге

- `WASD` — движение;
- `Shift` — бег;
- мышь — камера;
- подойти к порталу — перейти в приложение;
- `Enter` в поле чата — глобальный чат.

## Управление в survival

- `WASD` — движение;
- `Shift` — бег;
- мышь — камера;
- `ЛКМ` — добыча;
- `B` — включить/выключить строительство;
- `R` — повернуть строительный объект;
- `ЛКМ` в режиме стройки — поставить объект;
- `1-9` — выбрать слот hotbar;
- `I` или `E` — показать/скрыть Inventory;
- кнопки крафта справа снизу — создать предмет.

## Как добавить новое приложение

1. Создай папку:

```text
apps/new_app/
```

2. Добавь файлы:

```text
ico.png
index.html
client.js
```

3. В `index.html` подключи общий стиль, общий клиент и свой клиент:

```html
<link rel="stylesheet" href="/shared/style.css">
<script src="/shared/common.js"></script>
<script type="module" src="./client.js"></script>
```

4. В `client.js` вызови:

```js
await window.AppCore.init('new_app');
```

После перезапуска страницы каталог сам добавит портал.
